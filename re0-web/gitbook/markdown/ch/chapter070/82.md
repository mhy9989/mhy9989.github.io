# 『对叛乱者的猜测』

------

「──最终，抓住我是为了封锁德拉克洛伊领地的行动吗。果然，罗兹瓦尔，你找了个好助手啊」

「……虽然已经纠正过很多次了，拉姆是我的仆人啊」

考虑到所处的情况，塞琳娜异常沉着的发言让罗兹瓦尔无奈地苦笑着回答。

在两天前，他们拜访了塞琳娜的德拉克洛伊领地，并请求她在佛拉基亚帝国寻找失踪的昴和雷姆，为他们提供帮助。

塞琳娜是上级伯爵，她在帝国内颇有人脉。只要描述成「黑发黑瞳的少年，无论如何都会引人注目」，他们就有望从她那里获得准确的信息。

即使他们不知道被送到了何处，他们也无法想象昴会安分守己。所以从这个意义上讲，他们对她的期望很高。

因此，这样的结果在某种意义上是必然的，同时也是出乎意料的。

「拉姆，我还是问一下，这次行动的原因是……」

「──夏美・施瓦茨」

「就是这个了啊」

从拉姆的嘴里吐出的名字，对罗兹瓦尔来说也是无法忽视的。

那是被送往帝国并孤立无援的菜月・昴向能理解那个名字意义的人发出的求救信号──罗兹瓦尔和拉姆也立刻领会到了他的意图。

我就在这里，昴做出的这个决定，不同寻常地只有值得称赞的成分。是的，昴的行动本身并无过错。

问题在于，这个求救信号即狼烟所升起的『那边』。

正因如此，拉姆采取了近乎突袭和暴行的手段。

这样的结果，就是塞琳娜被拉姆用魔杖逼迫，坐在了沙发上。然而，这不能完全归咎于拉姆的暴走。

因为──

「──与反叛军一同的『黑发少女』，夏美・施瓦茨是你们要找的人吗？」

「严格来说，对拉姆来说那是附赠品」

「但对罗兹瓦尔来说并非如此。而对罗兹瓦尔来说并非如此，对你来说应该是非常重要的事情」

拉姆被杖指向，行动受限，她闭紧嘴巴，听着塞琳娜的话。

尽管行动受到限制，塞琳娜的思维仍然敏锐。即使在这种情况下，她也不失镇定，基本上了解了我们的情况。

──『夏美・施瓦茨』是菜月・昴用来自称的化名。

准确地说，这个名字还有更复杂的含义，但只要推测昴为何要在帝国使用这个名字，他赋予这个化名的意图便显而易见。

罗兹瓦尔本来也希望尊重这个意愿──然而，

「竟然，是与反叛军一起……」

虽然能理解使用化名的意图，但狼烟升起的方向却出乎意料。

本来在帝国潜入期间爆发的叛乱就是一个令人烦恼的问题。如今这个问题不仅没有解决，反而如同绽放出毒花般恶化。

不论是好是坏，情况不按照他们的预想发展，实在是典型的昴作风。

「拉姆，你是不是也受到了昴那家伙的坏影响啊？」

「这真是岂有此理。拉姆的行动，哪怕一丝一毫也不会受到那个巴鲁斯的影响」

「可是，事到如今，这种局面不正是因为他的行动而产生的吗？」

「罗兹瓦尔大人，请和塞琳娜女士谈谈」

一旦事情变得不利，她便巧妙地改变话题，并将主导权推给别人。

拉姆大胆的谈话技巧让罗兹瓦尔眯起了一只眼，他重新面对塞琳娜。

目前，罗兹瓦尔和拉姆被召唤到了她的办公室，在这里，他们亲自听到了她提起『黑发少女』。

作为寻找的目标，昴和雷姆的特征已经告知了她。因此，对她来说，只关注『黑发』部分的话题应该只是一个印象而已。然而，对罗兹瓦尔他们来说，这是他们不能忽视的意义。

拉姆采取了制约行动，就在塞琳娜主张「作为高级伯爵，我也必须对城郭都市发生的问题表示一些意见」之后。

胆大的塞琳娜，由于她没有在办公室安排护卫，这也招致了灾祸。

私人兵力正在房间外的走廊待命，他们并没有注意到室内的紧张气氛。当然，只要塞琳娜一声令下，他们会立即冲进办公室。

「那样的话，我会把塞琳娜女士撕成八片」

「哦，真可怕。我喜欢你这种既不是威胁也知道是认真的表情。罗兹瓦尔，你呢？」

「她是我的侍从，而不是伴侣……关于那个话题就此打住吧。比起这个，我们三人之间的关系能够恢复到和平状态，这才是我们想要努力的方向吧」

「你说『我们』，真是可爱。总是把一切当作别人的事的你，现在竟然试图承担起问题，给人一种新鲜的感觉」

塞琳娜含着笑对这番话，罗兹瓦尔眯着一只眼叹了口气。

确实如她所说，由于拉姆的行为，他不能再将问题当作与己无关。这不仅仅是主仆关系和责任所在的问题，还涉及到感情。

此外，拉姆的行动确实是有些仓促，但同时也可以说是因为急功近利才导致了这种过激行为。

毕竟，如果拉姆没有制约住塞琳娜——

「你引以为傲的『飞龙队』会在瞬间将反叛军烧成灰烬」

「——」

听到罗兹瓦尔的话，塞琳娜・德拉克洛伊微微眯起了眼睛。

德拉克洛伊伯拥有的『飞龙队』——那是在佛拉基亚帝国内驯服了大量飞龙，并且能随心所欲地飞行的帝国最强攻击部队。

原本就已经具有驯服不亲人的飞龙的『飞龙操控』的技术，飞龙骑士在战场上表现出了压倒性的战斗力。就连拥有飞行能力并且能够完全封锁大部分对手的罗兹瓦尔也明白。——控制制空圈是掌握战局的终极手段。

因此，在佛拉基亚帝国，拥有优秀飞龙骑士的家族更容易取得战果。

德拉克洛伊伯从上代开始就在这方面表现出色，塞琳娜・德拉克洛伊时代被认为拥有最强的『飞龙队』。

如果她真的全力动用『飞龙队』，大部分的反叛军简直不值一提。

「出乎意料地，你们要找的人现在已经成了反叛军的一员。动用我的军队对你们来说是不利的。但是，在这里我们只是相互对峙，这也不是办法吧？你打算怎么解决这个问题？」

「塞琳娜女士，您的家人呢？」

「寻找亲属是没有意义的。自从被罗兹瓦尔拒绝之后，就一直没有找到合适的人选」

「罗兹瓦尔大人？」

「别把玩笑当真。总之，王国和帝国的大贵族之间这种关系，在两国关系上是不可能的。塞琳娜，现在不是开玩笑的时候」

即使不考虑罗兹瓦尔和塞琳娜的关系，现在的拉姆也不能说是有耐心。

虽然一眼看上去不太明显，但拉姆对普雷阿迪斯监视塔的事件深感懊悔。失去了对昴和雷姆的监视，导致他们离散，她责怪自己的失职。

这次极端行动背后，她的焦虑毫无疑问也是其中一个原因。

「不要刺激拉姆，我们双方都寻求一个好的落脚点吧」

「双方的好落脚点，是吗。那么，你们具体希望得到什么？」

「——」

「没有确定目的地就让飞龙起飞的傻瓜是不存在的。那样会消耗大量燃料，骑手也很珍贵。你说的徒劳，正应该是这个。你应该明白我的苦衷吧？」

撑着脸颊，塞琳娜像是在试探罗兹瓦尔的反应。

有昴参与的反叛军，再加上雷姆很可能也在那里，拉姆急切地做出判断也是情有可原。即使责怪这件事，时光无法倒流，也是毫无意义的。

要是有心，他们完全可以伤害塞琳娜，在混乱中逃离领地。

但是，那样就放弃了最初的目标，只会制造更多敌人。即使是力量决定一切的帝国，士兵们也不会为被欺骗而高兴地接受主人被击败的事实。

削弱拥有『飞龙队』的德拉克洛伊伯爵的力量，或许对反叛军来说是件好事，但这种行为与罗兹瓦尔的友情是否相称，他心中也存有疑问。

无论如何，如果为了实现悲愿而需要牺牲，罗兹瓦尔也会毫不犹豫地牺牲塞琳娜的生命。

但是，如果没有必要牺牲，他并不想故意这么做。

因此，在这里罗兹瓦尔对塞琳娜提出的建议是——

「——怎么样，你愿意和我们一起加入反叛军吗？」

「啊？」

「这样一来，你和我们之间的矛盾就会化解。如果目标相同，拉姆也就没有理由再对你挥舞魔杖了。双方都能和平解决问题，何乐而不为呢？」

双手合十在胸前，罗兹瓦尔这么说，让塞琳娜目瞪口呆。

从她的眼中闪过的是惊讶、愕然，还有一丝好奇心。从激起她兴趣的角度来看，这并不是完全离谱的言论。

罗兹瓦尔凭借这点，努力拉近他与塞琳娜的关系。

「你说双方和平解决问题，但这只限于目前的情况。你的邀请最终会导向内战。我的领地已经得到皇帝陛下的保证，为何要无谓地挑起纷争呢？——」

「——皇帝陛下让你感到满意了吗？真的是这样吗？」

理智有条理地说出自己立场稳固的塞琳娜，然而罗兹瓦尔却打断了她的话，直视着她的双眼。

作为帝国的上级伯爵，表面豪放磊落的塞琳娜也是个擅长权谋的人才。正因如此，这个话题必须要在直视对方眼睛的情况下进行。

『和别人说话的时候，一定要好好看着对方的脸说话哦！』

突然，这样的话语在罗兹瓦尔的脑海中闪过。

在这种意想不到的情况下，被这句话激励着——

「这两天我并非在这里安然度日。在你忙碌的时候，我尽量留意了周围的情况。」

「原来你是在不陪我喝酒到天亮的时候，用心去偷听别人的话啊。」

「我建议你还是改掉每天喝酒到天亮的习惯，考虑到你的工作量。」

虽然招待客人的日子和平时的情况有所不同，但作为老朋友，他还是希望她能适量饮酒。

无论如何，在顾虑塞琳娜的肝脏健康之余，罗兹瓦尔展示了他偷听到的成果。这个德拉克洛伊领地秘密面临的困境，那就是——

「前年的未遂暗杀事件，传闻其中有『九神将』的参与，这件事情带来的影响似乎相当长久。」

「……」

「而那位实施犯并丧命的『九神将』是个骑飞龙的人……而且据说是你的部下。名字是……」

「——巴罗伊・特梅格里夫。」

塞琳娜用低沉、平静的声音说道，打断了罗兹瓦尔的话。

她依然用手撑着腮，但她的眼中的热情已经改变。原本似乎对这种情况感到愉快的神情消失了，取而代之的是一种宛如无风之湖水面的静谧。

保持那双宁静的眼睛，她看向斜后方的拉姆，

「拉姆，放下魔杖。就算没有它，我也会跟你们谈判的。」

「可是……」

「拉姆，就像塞琳娜说的那样去做吧。她不会违背她的话的。」

罗兹瓦尔举手制止，拉姆勉强放下了魔杖。

当然，她还是保持着随时可以重新做好准备的姿势，但塞琳娜并没有责怪她的警惕。只是，她将脸转向窗户，看向那里展开的景色——不，是抬头仰望天空。

飞龙展翅翱翔，自由地在空中飞舞。

「你觉得这片天空是自由的吗？」

「从你的口气来看，似乎并非如此呢。」

「帝国的水、土、空气，乃至人的血肉，都是皇帝陛下的财产。这甚至包括天空。——是谁泄露的？」

「要限定某个人的话，恐怕有点困难。毕竟，这是大家都在想的事情。」

罗兹瓦尔耸耸肩，塞琳娜简短地回答说：「是吗」。

她提到的名字——巴罗伊・特梅格里夫，曾是『九神将』之一，也是一年半前在帝都发生的皇帝暗杀未遂事件的实施者。

他原本是塞琳娜在德拉克洛伊领地内重用的人才，凭借非凡的实力和卓越的飞龙骑士技艺，从草根一步步晋升成了一将，似乎是一个极具潜力的人才。

当然，巴罗伊对皇帝翻起刃头，也让塞琳娜的立场恶化。

虽然没有被剥夺上级伯爵的地位，但在帝都的声誉受损，她的飞龙队失去了期待，也未给予挽回的机会。

更糟糕的是，取代巴罗伊成为『九神将』的人选——、

「——『飞龙将』玛德琳・艾沙尔，简直就像是在挑衅般的人选呢。」

「没有军队经历，凭借宰相的推荐成为一名将军的人物，竟然是令人惊讶的龙人。对于能够操纵飞龙的龙人来说，没有了『飞龙操控』的秘术，培养骑手的飞龙骑士就显得多此一举了。」

「那样的话，拥有飞龙队的塞琳娜大人的立场就没有了呢。」

「哎呀，别小瞧我的飞龙队哦。不需要骑手也有优缺点。确实可以省去培训的麻烦，但没有骑手的飞龙在战术上只能依赖本能。而飞龙骑士在战术理解和应用方面有着优势。」

「————」

「当然，那个龙人似乎只是用暴力的数量来压倒一切。」

佛拉基亚的秘传技法『飞龙操控』——关于如何驯服凶暴且不亲近人类的飞龙的技巧，即使罗兹瓦尔也没有完全掌握。但他听说要驯服一头飞龙，需要一个有才能的飞龙骑士全心投入其中。

换句话说，飞龙骑士必须与一头飞龙成为一对伙伴。

相比之下，不花时间就能控制无数飞龙的『飞龙将』对以飞龙队为优势的塞琳娜来说，无疑是天敌，就像眼皮上的痈疮。

「巴罗伊曾是我长期信赖的亲信。既然他背叛了皇帝，甚至对皇帝翻起刃头，责任自然会落到我身上。但是——」

「是否屈辱于此，又是另一回事了，对吧？」

「——。你总是善于触动我内心深处的话语。」

「甚感抱歉。」

拉姆用平静的语气回应并鞠了个躬，塞琳娜深深地吐出一口气。

塞琳娜并没有否定罗兹瓦尔和拉姆的指责，他们各自用不同的声音追问。从拉姆放下魔杖的那刻起，塞琳娜就已经做出了某种决定。

这表明她并不是在被威胁之下说出真相，而是以平等的眼光来进行对话的姿态。

换句话说——、

「——我本来并不打算在你们在这里的时候搅动风波的，明白吗？」

这么说着，塞琳娜・德拉克洛伊触摸着脸上的白色刀痕，獰猛地笑了。

那充满野性的笑容，正如年轻的罗兹瓦尔初次遇见她时所见，也如同在从父亲那里夺取家督之前表现出的雄心勃勃的笑容。

然而，这一次她的笑容所指向的，不是那个因女儿从自己手中夺走家臣信任而眼中显露迷茫的父亲，而是统治这片广袤帝国的可敬的存在——。

「——原来如此，塞琳娜，你是」

看到那个笑容，罗兹瓦尔终于意识到自己的误解。

本来他打算通过明确德拉克洛伊领地的立场，引导塞琳娜的想法，以避免拉姆草率行动造成的局势恶化，但这是个错误。

根本不需要引导她参与叛乱。

『黑发少女』参与的叛军崛起，对于她来说，仅仅是一个契机。塞琳娜已经做出了决定，并尽她所能地考虑到了罗兹瓦尔他们的立场。

她本来是想尽量不让毫不知情、前来寻人的旧友卷入其中，表现出了她独特的关心。

然而，罗兹瓦尔他们却毫无顾忌地闯入了对方的阵地。他们甚至没有察觉到魔石已经点燃，还在向其注入魔力。

换句话说——、

「我已经尽量小心了。这次你们就该怪自己娶了个懂事的媳妇。或者说，你当年拒绝了我的求婚就是报应吧。」

「塞琳娜」

「放心吧。只要揭露你的身份，人们就会怀疑你与王国内部有勾结。那样一来，发动两线战争根本就不是明智之举。如果要战斗，会选择有胜算的战斗。」

谈话的主导权被夺走，塞琳娜与罗兹瓦尔的局势发生了逆转。

既然立场重叠，罗兹瓦尔他们没有理由伤害塞琳娜。而塞琳娜则拥有处理有特殊情况的罗兹瓦尔他们的余地。

「罗兹瓦尔大人，难道说……」

「啊，我们就这样踏入了猎物的领地呢。——塞琳娜，如果我们没有提出来的话呢？」

「那时候我会适当款待你们几天，在内乱全面爆发之前把你们送出去。但是，既然被老朋友邀请一起战斗，就别无选择了吧。」

「真是无耻……」

究竟是打算响应这次叛乱，还是另有计划采取某种行动，尚不明确。但塞琳娜已经下定决心要反抗。

是因为皇帝冷酷的行径引发的愤怒，还是与她亲信丧命的事实有关，这一点并不清楚。

只有一件事可以肯定——、

「文森特・佛拉基亚皇帝陛下，在您的安定统治中，积压了许多不满的人。一旦大火熊熊燃起，火势将迅速蔓延。」

「……你对这火势有所了解吗？」

「谁知道呢。反过来说，难道你不觉得自己有所了解吗？」

含笑的塞琳娜似乎在暗示，她还有罗兹瓦尔所不知道的底牌。

与此同时，罗兹瓦尔也心知在这陌生的土地上能够发挥多少力量尚不清楚，但他知道，若放任不管，事情绝不会简单了事。

他有办法应对各种困境。关键是，在那些困境中，他会遇到谁，有多少愿望要实现，并将影响传播开来。

也许——、

「——让整个帝国燃烧起来，也是可能的吧，昴君。」

如果真的是那样的话，罗兹瓦尔应该如何行动呢？

无论如何，为了实现他的悲愿，他必须带着齐全的队伍回到王国。即使不是这样，他也和昴有约定。

虽然是单方面强加的，但昴不会违背约定。对于决心拾起所有残缺未来的他来说，这是难以接受的事情。

即使抛开和罗兹瓦尔的约定，他也会挣扎着伸出手臂。

那么至少，罗兹瓦尔只需要行动，让试验次数减少。

只要不让昴察觉到自己的愿望并未实现，那就没问题。

「那么，多亏了拉姆让我的生命陷入危险，现在我们之间的隔阂消失了。今天就不再工作了，干脆开瓶庆祝的酒吧。」

「虽然是举着杖的拉姆说这话，这样做真的没问题吗？」

「我们都需要为了相同的目标增强团结力。所以，喝酒增进友谊。想想我们的对手，这是再自然不过的事情了。」

塞琳娜若无其事地忘记了刚才的冲动行为，平静地邀请拉姆喝酒。

坦率地摊开心意，塞琳娜明朗地表现出与罗兹瓦尔他们无需藏匿的状态，拉姆罕见地感到困惑。

尽管如此，他们并不认为塞琳娜会在此时用花言巧语欺骗罗兹瓦尔他们，做出捅刀子的愚蠢举动。

如果那样做，罗兹瓦尔在帝国的自由将会和塞琳娜为实现目的而付出的战力，甚至她的生命一同陷入危险。

「那么，你是打算等待火势升起呢？还是成为火势的一部分？」

「自己的立场我还是很清楚的。没有必要急躁，时机快到了。——难道，你心中的想法是安分守己的性格吗？」

「……那可不是」

「我也是」

尽管彼们心中想象的『火势』各不相同，但对于这一情况的心情似乎是相似的。

罗兹瓦尔在思索着这个事实，双臂交叉。与此同时，塞琳娜平静地走向她的饰品柜，开始挑选要喝的酒。

「罗兹瓦尔大人，关于城郭都市」

「就地理位置来说，从东方进攻的爱蜜莉娅她们要近一些。他们那边也会听到同样的传闻，如果能够汇合的话，应该会这么做。当然，这对你来说可能会感到焦躁」

「……不」

拉姆摇了摇头，将内心的焦躁压抑住，回答道。

实际上，她应该渴望尽早与雷姆重逢。如果在城郭都市听到了关于昴的传闻，那么雷姆很可能就在那里。

另一方面，即使罗兹瓦尔使用了飞行魔法，如果空手而归的话，也会受到很大的打击。

在广阔的帝国寻人，需要某种程度上压抑内心的理性。

「正因为如此，我们应该等待。就像她所说的，等待火势升起」

「――好的」

在稍微犹豫和踌躇之后，拉姆勇敢地点了点头。

――几天后，塞琳娜・德拉克洛伊所说的『火势』升起了。

皇帝，文森特・佛拉基亚的子嗣为首的反叛军，其势头一举席卷了整个帝国，传播到各地的反叛军们。

在此过程中，最早宣布加入反叛军的是德拉克洛伊上级伯，据说在勇敢的女领主身边，有一位英俊潇洒的陌生人和一个可爱的随从相伴。
