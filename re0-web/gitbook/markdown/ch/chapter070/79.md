# 『绯色的大笑』

------

--普莉希拉突然出现，掀起了一股言灵旋风。

「——」

瞬间，明确的沉默和空白支配了整个大厅，每个人都感到了时间仿佛停滞的错觉。

虽然是无法听错的话语，但似乎无法理解其意义。这种印象就像无法从声音中正确地理解意义，需要在脑海中寻找「哈哈韦」的含义一样。

然而——

「——？尤尔娜是普莉希拉的母亲？」

当爱蜜莉娅奇怪地歪着头，发出了另一个「咔萨玛」的音响，随即「哈哈韦」和「咔萨玛」相结合，实体化了「母亲」和「母上」的概念。

尽管感到自己的头脑有些混乱，奥托还是比较关注这两个人 —— 普莉希拉和尤尔娜。他看着呆若木鸡、动弹不得的尤尔娜，以及沉着冷静的普莉希拉。

这两个人的发色、瞳色都不同，人种也不同。当然，对于混血的情况，父母血统的遗传比例因人而异，但从外貌特征上看，普莉希拉和尤尔娜几乎没有相似之处。

如果非要说，那么两人都是拥有迷人美貌的人，但是——

「如果只因为这个而视她们为家人，那这个世界就充满了大家庭。」

奥托得出了这个结论，控制住了自己的困惑。

尽管如此，听到普莉希拉和尤尔娜的关系后，有些事情也变得更加合理了。普莉希拉苛刻的态度和性格，确实非常符合帝国式的特点。

「虽然感到了望向妾身的不愉快目光，但也罢。」

当奥托读懂普莉希拉的目光并被她警告时，他不禁屏住了呼吸。然而，普莉希拉却没有顾及奥托，转而望向爱蜜莉娅。

「半魔，你的话毫无差错。她居然成为了『九神将』之一，这真是奇怪的命运啊。」

「是啊，我也很惊讶。但是，被你这么一说，我发现我们两个真的很相似。」

普莉希拉用手中的扇子托起下巴，爱蜜莉娅点头响应了她的话。

之前，奥托内心否定了她们外貌上的相似之处，但爱蜜莉娅却做出了完全相反的判断，这让他感到困惑。不过，先将这个问题搁置一下。

问题在于——

「——等一下。」

普莉希拉的话让尤尔娜感到意外，并停下了说话。她闭上眼睛，从表情上打消了之前的烦恼，接着说道：

「虽然有些惊讶，但你的话不值得一提。为什么你会把我当成你的母亲呢……」

「别用这种无聊的借口。如果你想对妾身下手，你必须改变自己的灵魂本质。无论你的形态如何改变，妾身的绯红瞳孔都会看穿你的诡计。」

「……」

尤尔娜假装镇静的表情再次变得僵硬，普莉希拉的断然回答让她明显感到不安。

普莉希拉的话很抽象，而且没有任何向别人解释的意愿。因此，很难理解她的确切意思，但是这些话对尤尔娜来说显然产生了致命的影响。

普莉希拉锐利的眼神和舌头使尤尔娜明显感到不安。

然而，在这个场合下，感到强烈不安的不仅仅是尤尔娜……

「——真是个笨蛋。」

他们的惊叹清晰可见，这种微妙到几乎听不清的声音，对于说话的人来说却是很明显的。奥托听到了鬼面的一句话。

那个人物，埃布尔，给人以冷静而又冷酷的印象，没有任何动摇。他问道：

「普莉希拉，你是认真的吗？尤尔娜・米希格蕾就是桑德拉・贝尼迪克特吗？」

「嗯，原来你也不知道啊，埃布尔。没关系，这也是理所当然的。除了自己申报之外，没有其他途径可以知道。」

「但是普莉希拉，即使没人告诉你这个消息，你也发现了这件事吗？」

「别打断谈话。半魔，安静点。」

「你不用这样说……」

普莉希拉淡然地回答了埃布尔的问题，然后凝视着想插话的爱蜜莉娅，迫使她闭嘴。

总之，国王选举的人员和帝国的人员都不了解情况，他们的交谈被搁置了。虽然奥托也对普莉希拉的内幕感到兴趣，因为他们都是王位继承人的候选人，但此刻优先处理这个孩子的问题。

「别再谈论那不期而遇的事了。现在，这个孩子的问题更为紧迫。」

即使气氛有所变化，加菲尔依然在威胁，瞪着他碍事的目标：被尤尔娜和米迪娅姆保护着的路伊。正如加菲尔所说，如果不能解决她，他们将无法前进。

当然，奥托的结论是束缚和打倒路伊，夺取她的自由，这一点是毫不动摇的。

「别让她再呼吸。但是，把她绑起来推到一边是我不能容忍的一条线……这是『蒂诺斯的手脚远离』的故事。」

「那可能是兄弟不希望听到的话。」

「哦？」

奥托和同一立场的加菲尔对他的话语感到不满。

插话的人是普莉希拉和一起上到最上层的阿尔。他那独臂的奇异外表一如既往地让人感到不舒服。

「首先，我要向姑娘道歉。我们去了卡欧斯弗莱姆，却没能带回我们的兄弟，真的很抱歉。」

「阿尔……谢谢你的道歉，但我不是唯一一个因为见不到昴而感到失望的人。最失望的应该是碧翠丝。」

「啊啊，那我也向那个孩子和其他人道歉。」

阿尔低下了头，为昴不在的事情道歉。

尽管阿尔平常表现得轻率，但他的道歉之意似乎是真诚的。然而，真诚的道歉并不总是会带来原谅和好印象。

「相反，轻率的道歉只会增加被利用的可能性。如果有真正想道歉的心，我认为最好保持沉默。」

「你真是太苛刻了。不过，我已经说过了，处理这个小家伙的事情并不是我们的……菜月・昴绝不希望听到的话。」

「那么，你们为什么会这样！一直在说些随便的话……」

「我也是因为要杀掉那个小家伙，结果被兄弟阻止了。」

加菲尔激动的话语被阿尔干巴巴的话语所压制。阿尔耸了耸肩，用下巴指了指路伊。

「你们也是这样的。我想要杀掉他也很自然吧？尽管这样……」

「那昴拒绝了吗？」

「说实话，我差点不敢相信。本来只是因为不知道这个小家伙是谁而抓住他的，但他却是大罪司教。即使这样，菜月先生还是为他冒险保护他。我原本就知道兄弟们的想法有些奇怪，但这次的情况实在超出了我的想象。」

阿尔的目光透过头盔直视着路伊。路伊微微咕哝了一声，但并没有躲闪，而是直接对视。

那双蓝色的瞳中闪耀的光芒，不是卑微也不是贫弱。

奥托审慎地考虑着阿尔的陈述，如果是了解昴的团队成员，他们会同意阿尔的话不是口出狂言。奥托认为昴没有对路伊下手的原因是『软弱』。

昴和爱蜜莉娅身上所拥有的『软弱』既是弱点也是优点。

一旦失去了这个优点，就无法再挽回，虽然处理起来有些困难，但奥托并不想失去它。

「这种没有『软弱』介入的判断，由我们周围的人做就好了。」

「我知道，哥哥。但现在已经很难了。毕竟我们正在这么多人面前说话。再也回不去了。」

「……奥托大人。」

法兰黛莉卡踏着鞋子发出了踏实的声响，她向奥托的侧脸呼喊。她美丽的眼睛中映照出的是对于此时此刻发生的事情的担忧和不安。

奥托也感觉到，在这里提到路伊的身份是一种错误。不，说出这句话，已经说明路伊的身份已经被认可为相关人员。

塔莉塔和米泽尔妲所得出的结论，在城郭都市中被普遍接受。

因此——

「那么，如果放任她不管呢？」

「当然，如果她做什么事情，我也不会手软。但是……」

「路伊一直在保护昴。即使和我们在一起，也从未做过坏事。将来也不会的！」

中间接过阿尔的话，米迪娅姆发出了拼命的声音。

她的话只是一种希望，迄今为止路伊的行为并不能保证未来的行为。这正是这个问题的最大焦点。

「我看过主人使用誓约咒印的样子，我也可以做到……」

「……不要这样。虽然我也一瞬间认为这是一个好主意。」

佩特拉认真地提出了一个建议，但奥托却拒绝了。

佩特拉所说的『咒印』是为了让目标遵守承诺而约束灵魂的东西。这是罗兹瓦尔在自己身上刻下的，如果被破坏了，他将失去生命。

在『圣域』和旧罗兹瓦尔宅邸发生的事件中，罗兹瓦尔刻下这个咒印是为了表示他向阵营的投降。

正如字面所示，『咒印』明显是一种『诅咒』。

这是一种迷信般的想法，但有人说诅咒最终会回到使用者身上。如果一味为了方便而不断用诅咒约束他人，最终诅咒也将烧尽自己的灵魂。

佩特拉不想承担这样的厄运，即使实行并刻下了约束路伊行动的咒印，它也没有保险以外的意义。

「因为我们不知道约束到什么程度才能使我们安心」

大罪司教的权力，其全部情况是没有人知道的。

因为不知道什么隐藏的东西会飞出来，所以即使约束了什么，奥托对路伊的警惕也无法解除。除非他夺走他的生命。

那么，没有可能导致沉迷的咒印，警惕是更稳定的。

「很抱歉，但这是我的结论」

「奥托哥啊！你就这样放任了？大罪司教啊！？」

听了奥托的想法，加菲尔大声尖叫。

虽然知道自己是正确的，但仍感到空气中有一种被强行封殺的感觉。奥托也想体会弟弟的痛苦心情。

「但是，在这里讨论是得不到赞同的。在现状下，还有待听取的话语，我不想与大家激化矛盾。」

「所以，如果要实际行动，应该在听完对方的意见后再说。」

「我不打算做这样危险的事，但是我会把这看作是赞美的话，普莉希拉.」

奥托勉强回应了普莉希拉精确地说到心坎里的话。

无论如何，尽管很不满，回答加菲尔的结论是事实。

这个城市有话语权的埃布尔和对待路伊的态度不妥协的『修德拉克之民』，没有办法摧毁他们的想法。

作为大罪司教，没有根据驱逐路伊。

如果这不通用，那么除了实际行动，我们没有其他途径。但是，即使杀了路伊，『暴食』的权力影响也不一定会消失，此后帝国的全部敌人，抱着瑕疵而做的价值是否足够？

「这样争吵不会让我们的不安减少一点。是吧？」

「――――」

牙齿咯响的加菲尔和眯着一只眼睛的奥托。随着话语的展开，眼神晃动的爱蜜莉娅的紫瞳眼睛也晃动了。她的胳膊里，被抱着的碧翠丝叫她的名字：「爱蜜莉」。

「不好意思，碧翠丝……小姐。我知道你在担心我。」

「……如果你真的知道，那就好了吧。」

眼神和语言在近旁交换，碧翠丝低下了她圆润的眼睛。顺着她的感受，点了点头的爱蜜莉娅把目光朝着路伊和米迪娅姆的方向。

那个少女像姐妹一样回应了她的目光。

「那个孩子……路伊可能很危险。正如奥托大哥们说的那样，我知道这件事。」

「……是的，我知道。」

「但是，你没有看到路伊做出危险或者坏事的地方。路伊也爱着你，还有昴？」

「是的，他想要保护我们。真的，不是假的。埃布尔、阿尔、普莉希拉、尤尔娜，他们都知道，对吧？」

在被直接凝视着的情况下，米迪娅姆尽力细心选择着词语。她在选择说话的同时，向一起从魔都回来的人们寻求同意。虽然埃布尔和尤尔娜还没有从刚才与普莉希拉的交流中站稳脚跟，但阿尔摇摇头，塔莉塔默默点头。

「恩，路伊的确是想要保护大家的。特别是她特别珍爱昴。我是这样认为的。」

「没想到『幼女使』的形象真的这么鲜明。」

「……这个别名，贝蒂并不怎么喜欢。请注意你的言辞。」

不满的碧翠丝低沉的声音略微缓和了她之前的战意。关于路伊的处理，碧翠丝已决定委托给爱蜜莉娅。

尽管是同样的结论，爱蜜莉娅以温和的方式整理了奥托和碧翠丝提出的话。

因此，爱蜜莉娅朝着路伊的方向看着，说：

「你曾担心昴，我不相信那是假的。所以，我也想像这个孩子一样，忠诚地相信你。」

「哦，爱蜜莉娅小姐，那个家伙，」

「加菲尔最开始对我们很恼火，但现在我们是好朋友，不是吗？」

爱蜜莉娅的话有点卑劣，但加菲尔和路伊的立场和情况都不同。然而，当爱蜜莉娅如此断言时，感到难以反驳的事实是事实。

情况也在帮助加菲尔，他痛苦地咬紧了自己的脸颊。对于加菲尔的反应，爱蜜莉娅低声道歉，说：

「这可能是件困难的事。但是，我不想一开始就制造冲突，如果我们可以进行良好的沟通，那么从那里开始是最好的。当然，有时候也不得不制造冲突，但是……」

「……」

「我希望能和在这里的所有人友好相处。为了这个，我希望能清楚地表达我希望友好相处的愿望。因此，不如先放开我们握在一起的手。」

说着，爱蜜莉娅低头看了看自己怀里的碧翠丝。她的目光让碧翠丝微微睁大眼睛，然后轻轻点头。

爱蜜莉娅稳稳地把自己的身体向前倾。在尤尔娜面前穿过与她对视的加菲尔，来到保护着小小的路伊的小小的米迪娅姆面前。

然后……

「很抱歉，但是……我相信昴和雷姆，也相信他们相信的米迪娅姆们，因此，我希望你也能相信。」

「……啊，哦」

「对于这些一直很好地照顾你的孩子们来说，有人相信你是非常高兴的。」

说着，爱蜜莉娅轻轻伸出了她的右手。

左手抱着碧翠丝，跪下的爱蜜莉娅面向着路伊，引起了守护着路伊的米迪娅姆的注意，

「路伊酱」

她这样呼唤着。

不知是呼唤成了导火索，还是爱蜜莉娅的话对没有人性的大罪司教产生了某种影响，路伊犹豫着伸出手。

路伊的右手放在了伸出的爱蜜莉娅的右手上。

即使有担心爱蜜莉娅的势力会从这里掌握权力的可能性，奥托也认为自己应该蔑视这样的担心。

无论如何——

「啊啊」

「嗯，我也很担心昴」

握着手，爱蜜莉娅这样回答，对于奥托来说，也是对爱蜜莉娅阵营的结论。只不过是一个『保留』的结论。

但是，尽管处境相同，爱蜜莉娅说的话和自己说的话的意义是不同的，奥托为爱蜜莉娅感到骄傲。

「我可不会把眼睛从你身上移开的。」

对于如何对待路伊的问题，暂时做出了『保留』的决定。

然而加菲尔却依旧坚称自己无法完全接受这个结果，他对于刚刚与爱蜜莉娅握手的少女也说出了同样的话。

考虑到加菲尔的忧虑和警惕是很正常的，奥托并没有发表意见。

「毕竟我们不可能想象得出魔女教或者大罪司教会对改邪归正。」

奥托的干巴巴的现实思维得出了这样的结论。即使爱蜜莉娅和路伊之间发生了感人的、历史性的交流，他也认为这是不符合自己的风格的。

「光是击败白鲸和大兔，制裁大罪司教的『怠惰』和『强欲』，我也无法想象。」

但一个人在短短一年多的时间内多次做到了这一切，那就是昴。

虽然外界视爱蜜莉娅阵营的成就为真正的胜利，但阵营中的每个人都理解昴的贡献是巨大的。因此，他们不得不考虑到这种可能性。

昴到底又做出了什么令人惊讶的行为呢？

「我不想……」

如果那样的事情真的发生了，周围的人会怎样评价昴呢？这让人感到沉重。

虽然我觉得应该尽快让其他人了解昴的真正人性……

「能不能在讨论稍微平静下来后再说？」

这时，兹克尔举手发言，缓解了一下气氛。

他说：「虽然我也很感兴趣普莉希拉小姐和尤尔娜一将的关系，但还有几件事情需要跟埃布尔确认。」

「……好的。」

「嗯。」

在话题的矛头指向他时，一直保持沉默的埃布尔点头表示同意。

虽然他没有介入对待路伊的问题，但是对于之前的冲击，他似乎做出了一个临时的结论。听到兹克尔的呼吁，他注视着他的黑瞳，促使他继续讲下去。

「我们知道即使在尤尔娜一将的帮助下，魔都卡欧斯弗莱姆依旧不可避免地崩溃了……还有就是，昴遭遇了危险。但是，我想请教的是最近传播开来的谣言——」

「——谣言？」

「是的。——据传皇帝殿下有一位黑发的皇太子。」

兹克尔恭敬地点头，向埃布尔传达着这个消息。

这个谣言——关于作为皇帝文森特・佛拉基亚的儿子存在的传闻——已经传到了奥托的耳中。

据说，皇帝的儿子是叛乱军的领袖之一，也就是在这个城郭都市古拉尔聚集的一群人，反叛了皇帝。

不过，我在这个城市里并没有见过那个人。

「据报道，这个谣言是从东方传来的。虽然已经传播到了帝国的各个地方，但是这个消息的来源是……」

「正如你所猜测的一样，是从魔都传来的。——要威胁王座，需要力量。而力量只有在大义之下才能集结。」

埃布尔摇了摇头，双臂抱在胸前淡淡地回答。

虽然这个答案令人信服，但是从阵营的角度来看，试图威胁皇帝的位置本身并不是他们想要参与的问题。

奥托的焦点在于如何阻止爱蜜莉娅放弃曾经认识的人们。

正因为如此，他没有及时意识到他们可能会知道这个消息。

因此——

「……黑发的，皇太子。」

突然，沉思的低语是佩特拉说出的。

她在可爱的眉头之间皱起了一道细小的皱纹，显得若有所思。然后，犹豫不决地将这个想法说了出来。

那就是——

「那个，刚才米迪娅姆先生他们谈到的，是说昴变小了对吧。像我和碧翠丝那么小」

「确实是这么说的。虽然很难想象，但如果是昴大人的话，也有可能遭遇这种事情……啊」

「——！」

听完佩特拉的话，法兰黛莉卡整理了一下信息，然后瞪大了眼睛。几乎与法兰黛莉卡的发现同时，奥托也明白了佩特拉在想什么。

然后，他意识到自己的想法有多么离谱，以及这个令人诅咒的事实。

「难道说，传闻中的皇帝的孩子，就是昴吗？」

代替诅咒这种可能性的奥托，佩特拉从正面提出了这个问题。

她直勾勾地盯着埃布尔，鬼面后的黑眸接受了这个质询。埃布尔毫不动摇地，面对佩特拉的追问，轻轻地点了点头。

「没错」

「——怎么可能！」

对于简短的回答，佩特拉眼神变得锐利，正想提高声音。

然而，在佩特拉的反应之前，

「——哈哈哈哈哈！」

这样的发自内心的愉快的声音回荡在空气中。

这毫无顾忌的笑声破坏了大厅的氛围。然而，没人立刻反驳这种嚣张，因为那响亮的笑声是普莉希拉的。

她捂住嘴巴，用扇子遮住嘴角，尽量不让别人看到她笑得露出的牙齿，

「居然说那个平庸之辈是皇太子，真是太好笑了。啊，埃布尔，你这家伙，是想让我笑死吗？你的手法可真是变了不少啊」

「公主？」

「你也笑吧，阿尔。不，你是参与这个阴谋的一方。既然如此，让我大笑不止。这种小丑的职责，不得不说干得漂亮」

普莉希拉转向惊讶的阿尔，眼角微垂，赞美着他。她出人意料的反应让大厅暂时失去了热度。

当然，并不是说这样就消除了所有的混乱。

「呃，这是怎么回事？昴是皇帝的孩子，怎么可能呢？因为昴是从大瀑布的另一边来的啊」

「爱蜜莉娅，那是昴大人在开玩笑」

「是这样吗？那么，真的吗？」

「关于是不是皇帝的孩子这个问题，是假的。是谎言」

「哎？哎？哎？」

爱蜜莉娅一头雾水，目瞪口呆。

她的困惑也不无道理。总之，昴遭遇的幼儿化情况与埃布尔他们的目的恰好契合。

「确切地说，我认为应该说是利用了这个情况」

奥托同意法兰黛莉卡的说法，这是对情况的有效利用。

埃布尔他们在发动叛乱时，渴望一个貌似正当的理由。为此，皇帝的孩子这个幌子没有比其他更具说服力的了。

当然，以一个不存在的人物为旗帜也可能是一把双刃剑。

「既然菜月先生是真实存在的，那么双刃剑也可以妥善使用吧」

「那种说法，虽然无礼，但我很喜欢。那么，你也应该明白了」

「——虽然很不甘心」

本来，奥托他们应该对利用昴的状况感到愤怒。

然而，不得不承认埃布尔策划的这一计谋具有一定的价值。

原因在于——

「——找不到下落的昴大人，死亡的可能性大大减少」

「……是的」

法兰黛莉卡的理解，奥托沉重地点头同意。

爱蜜莉娅和加菲尔对奥托和法兰黛莉卡的想法反应迟钝。两人面露困惑，扭头问道：「这是怎么回事？」

「完全不明白这个对话……」

「虽然明白昴被加上了奇怪的头衔。但如果他被当作叛乱的首领，那不是更危险了吗？」

「不，尽管会吸引注意力，但生命危机或多或少会减少。这个『皇太子』的价值只有在他活着的时候才能发挥」

威胁文森特・佛拉基亚统治时代的不安分子『皇太子』。

这个叛乱军的正义理由对现任皇帝的支持者和敌人都具有利用价值。对叛乱一方而言自不必说，皇帝一方也更倾向于让他活着，因为这样更有用。

只有利用活着的『皇太子』才能削减叛乱军的势头，或者让他失去正义理由。

「也就是说，无论菜月先生被送到哪里，被谁控制，都不太可能会立即被杀。但代价是——」

「代价是？」

「……佛拉基亚的皇位争夺战，他将无法逃离这场大乱战」

接受恩惠就意味着承担相应的责任。

无论愿意与否，昴都被卷入了佛拉基亚帝国的大乱战，甚至被置于最高位置的御輿上。

正因为如此——

「——怎么样，无法不笑吧？」

普莉希拉提前得到答案，终于领悟到她恶作剧般地笑的原因。
