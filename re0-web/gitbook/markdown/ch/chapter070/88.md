# 『突破口』

------

——帝都鲁普加纳的包围战，从双方毫不保留的激战开始。

现在佛拉基亚的反叛军对国家抱有不满，而正规军则力图维护这个国家。本来，爱蜜莉娅等人没有必要参加这场堪称全面战争的战斗。

起初，爱蜜莉娅一行之所以进入帝国，是为了寻找失踪的人。

当然，他们潜入佛拉基亚，途中结识了许多人，与他们建立了感情和义理。

他们希望那些无视国家界限的幸福人们、以及那些愿意提供帮助的对象都能得到幸福。这一点是他们一行人所共同认可的。

然而，即使对这群毫无疑问是好心人的爱蜜莉娅一行来说，他们也有优先次序。

要实现最初的目标，而不是本末倒置地反射性地去帮助别人（爱蜜莉娅、加菲尔），他们的团队中还有能够进行冷静判断的力量（奥托、佩特拉）。

他们参加了被飞龙袭击的城郭都市的战斗，也是因为他们一直在追踪着寻找那个城市的伙伴的信息。

既然这个线索没有结果，那么按理说，爱蜜莉娅等人参加叛军攻击帝都的理由就根本不存在。

实际上，在听到帝国日益恶化的形势时，爱蜜莉娅一行的决心很坚定，他们对目标的关注仅受到了最小程度的干扰。

然而——，

「——那个孩子被『飞龙将』带走，那么她应该在龙人玛德琳・艾沙尔的住处，或者是那个家伙的主人的府邸里。」

站在城郭都市市政厅最高层房间前展开地图的埃布尔，他摸了摸遮住脸的鬼面，回答了爱蜜莉娅的问题。

听到埃布尔的回答，爱蜜莉娅低声说：「玛德琳的主人……」

—— 时间回溯到帝都鲁普加纳包围战的几天前。

原本应该与昴在魔都会合，然后返回古拉尔的爱蜜莉娅等人，却因计划出现差错而无法与刚刚苏醒的雷姆相遇，他们对于接下来的行动策略感到迷茫。

在魔都卡欧斯弗莱姆崩溃后，他们不知道昴的下落，却知道绑架雷姆的对方的身份，他们纠结于应该优先寻找哪个人。

爱蜜莉娅渴望得到判断依据，而埃布尔告诉她的就是开头的话。

「虽然不能告诉那个脾气暴躁的龙人，但他可是宰相贝尔斯提兹・冯达尔福饲养的一将。如果发生什么事，她会回到饲主的身边。」

「——」

「怎么了？有什么疑虑吗？」

「不，不是那个……只是觉得用『饲主』这个词说法，很不妥当。」

虽然爱蜜莉娅与玛德琳的交流很少，甚至可以说两人关系非常紧张，但她认为埃布尔现在的说法，如果让玛德琳听到，肯定会让她非常生气。

面对爱蜜莉娅的指正，埃布尔默不作声，眼神从面具的深处变得狭长。希望他能回想起自己的话，并对其表示反省。

无论如何，对于爱蜜莉娅他们所不了解的帝国情况，埃布尔的见解是非常有帮助的。

帝国各地的反叛氛围日益高涨，甚至连天空和空气的干燥程度都变得糟糕，这种感觉让希望事情尽可能发展顺利的爱蜜莉娅也不禁感到不安。

他们一直在寻找失踪的人，但在帝国的情况非常不妙。

「但是，如果玛德琳回到那个贝尔斯提兹的身边，雷姆也会……」

「如果雷姆在那里，那个家伙也会在！」

爱蜜莉娅顺着话题，此时正准备制定战略的米迪娅姆加入了讨论。

米迪娅姆竭力伸展着她的小身体以吸引注意力，与爱蜜莉娅的目光相遇，爱蜜莉娅的嘴角微微上扬。米迪娅姆的亲兄长也是被玛德琳带走的人之一，与雷姆在一起。

尽管肯定非常担心和不安，米迪娅姆却一点也没有表现出软弱的样子。

爱蜜莉娅觉得自己也应该向她的坚强学习。

「宰相……贝尔斯提兹・冯达尔福阁下，是一个非常优秀且有能力的人，被称为帝国的支柱对吧？」

说这话的是在爱蜜莉娅旁边，被米迪娅姆的勇敢所感动的佩特拉。

为了不让爱蜜莉娅独自承受压力，佩特拉一直陪伴在她身边。爱蜜莉娅歪着头问道：「真的吗？」佩特拉用手指碰了碰她的嘴唇，低声说道。

「帝国的支柱这个评价很高，但这不是指皇帝吗？」

「当然，帝国长期和平都是因为皇帝陛下的力量。但是，皇帝陛下的力量，也包括善于使用优秀部下的能力。」

「啊，是这样啊。即使皇帝非常强大，非常聪明，也不可能一个人完成所有事情。」

正在努力的爱蜜莉娅并不认为自己与佛拉基亚皇帝处于同一个层次。但是，她迟早需要达到同样的地位，而且她应该为此而努力。

然而，无论爱蜜莉娅努力到什么程度，她肯定会一直借助其他人的力量。

佛拉基亚的皇帝也应该是这样。

然而，在令爱蜜莉娅和米迪娅姆敬佩的同时，埃布尔似乎有着完全不同的看法。他透过鬼面看着佩特拉，眼神变得锐利。

「故意让我听这个理所当然的道理是为了什么？你认为我有空倾听一个小孩子的话吗？这时间可是宝贵的。」

「对不起，我说得有些拐弯抹角。其实，我只是想问这个 —— 如果是皇帝陛下的得力助手的话，宰相的住所应该是在皇都吧？」

「……」

佩特拉毫不畏惧埃布尔的锐利眼神，而埃布尔对她的质问保持沉默。

埃布尔可怕的目光，本应让人感觉仿佛被刀锋所指，然而佩特拉却丝毫未受影响。这足以证明她的心灵有多强大。

这让作为她的亲人的爱蜜莉娅感到自豪并怀着骄傲。

「你的疑问，我的回答是肯定的。他们必然被玛德琳・艾沙尔带走了……雷姆和弗洛普・奥克尼尔也将被带到位于皇都的贝尔斯提兹的住所。」

埃布尔似乎被佩特拉的目光压倒，低声回答道。爱蜜莉娅听到这个回答，想要称赞佩特拉真是了不起。

然而，佩特拉的表情却没有高兴起来，反而明显地叹了口气，

「……这感觉像是被安排好的，一点都不有趣。」

「诶？这是什么意思？」

爱蜜莉娅不明白佩特拉的反应意味着什么，她瞪大了眼睛。接着佩特拉的话语，她依然面带酸楚的表情，声音也变得低沉，

「我们和这个帝国发生的问题并无关系。我知道这会演变成一场非常大的战斗，虽然很担心米迪娅姆和乌塔卡塔这样的孩子们，但是……」

「――。嗯，没关系的。我明白你想说的。我们来这里是为了寻找昴和雷姆。」

爱蜜莉娅温柔地选择了词语，抢在似乎难以启齿的佩特拉之前说出了话。

爱蜜莉娅当然能理解佩特拉的不安和顾虑。如果条件允许，爱蜜莉娅也想和埃布尔他们一起合作，为所有相识的人提供帮助。

但是，如果只关注眼前的问题，她们就无法达到寻找昴等人的真正目的——

「咦？」

正考虑到这里时，爱蜜莉娅突然感到有些困惑。

正如她向佩特拉解释的那样，她们的目标是安全地带回昴和雷姆。所以即使很痛苦，她们也必须无视佛拉基亚发生的战斗。

虽然她是这么想的。

「但是现在，雷姆可能在皇都。那么……」

「――我们也必须去皇都」

爱蜜莉娅的疑惑消散，她和站在冰的另一边的佩特拉四目相对。

佩特拉早已注意到这个事实，所以她一直都是一副苦涩的表情。为了证实爱蜜莉娅的惊讶，佩特拉瞪了埃布尔一眼。

接着――，

「太卑鄙了。在我们提出来之前，你一直保持沉默。」

「如果按照道理来说，这都是理所当然的事情。你们却因为自己血液循环不佳，把我看作卑劣，这本身就是傲慢。」

「那么请告诉我。您说过『飞龙将』玛德琳要回去的地方有皇都的宰相那里，还有另外一个地方。但是，您一直在谈论皇都，却故意让爱蜜莉娅没有注意到那个地方。」

「――――」

「在我家乡，利用别人的无知来占便宜被称为无耻行为。」

佩特拉用一种尖刻的眼神盯着埃布尔，根据佩特拉的说法，爱蜜莉娅因为被误导而掩盖了信息的漏洞，皱起了眉头。

在埃布尔身边，正在仔细审视地图的米迪娅姆也瞥了一眼埃布尔，「埃布尔先生……」，她用提醒的语气看着他。

爱蜜莉娅深入想了一下，她能地理解佩特拉为什么生气。

把事情隐瞒起来当然不是问题，但是埃布尔所说的事情，如果爱蜜莉娅仔细思考，她也能意识到，即使埃布尔没有告诉她，她也应该知道。

当然，爱蜜莉娅并不打算在所有问题上都依赖他人。

「埃布尔也说时间很紧，对吧？那么这就是相互之间的事情，不要消极对待。」

「这说明你的傲慢。为什么我要专门为你讲解事情的道理呢？」

「如果你们需要我们的力量，请说出来啊。」

「……不合作的女人。」

埃布尔双手交叠，长长地呼出了一口气。

然而，爱蜜莉娅对埃布尔 的回答有了「是吗？」的不同意见。不是他们合作，而是埃布尔不喜欢和她们合作。

「是啊？因为我们接下来要在佛拉基亚参加最艰苦的战斗，我们的力量也会派得上用场吧。我们可是非常厉害的啊。」

「爱蜜莉，请注意说话方式。」

「我们可是非常可靠的啊！」

「非常可靠，尤其是爱蜜莉和加菲尔先生。」

爱蜜莉娅纠正了自己的说法，佩特拉也挺起胸膛说道。

爱蜜莉娅一行——加菲尔毫无疑问是最强的战士，爱蜜莉娅也相当有自信。奥托和佩特拉非常聪明，法兰黛莉卡则擅长体贴关心他人。

碧翠丝在与昴重逢之前无法发挥全部力量，正在休息中。但是，只要想到要保护她，就会激发起无穷的力量。

因此——

「我们在场的话，埃布尔的目的一定会更容易达成吧？可为什么只能用这种说话方式呢？是因为我们不想被认为是请求帮助吗？」

「――――」

「虽然罗兹瓦尔说『拜托了』这种话很拙劣……」

埃布尔和罗兹瓦尔一样，都不想让对方看到自己的弱点。

因此，他们总是试图引导对方说出什么。也许这种方式一直行得通，但是……

「在遭受非常惨痛的报应之前，最好停止这种行为。就像罗兹瓦尔被米迪娅姆、尤尔娜、兹克尔等人狠狠地打击一样。」

当宅邸和『圣域』卷入邪恶的计划中时，罗兹瓦尔包括爱蜜莉娅在内的所有人都遭到了沉痛的打击。

尽管爱蜜莉娅参与其中，但她也感到非常痛苦。虽然罗兹瓦尔已经训练有素，只是脸部肿胀，但对于埃布尔来说，情况可能会更糟。

即使不考虑埃布尔身体的问题，也是如此。

「我的话完全错了吗？如果是的话，那太丢人了，请说出来。如果那样，我也会道歉并考虑其他事情。」

「其他事情？」

「我并不认为你是一个什么都想不到，不能考虑未来的人。重要的是要一直思考，想到自己能想到的极限。」

以前的爱蜜莉娅总是很快放弃自己无法理解的事情。

她总是很快找个借口放弃，并依赖帕克的话。近来，她经常向拉姆和奥托等擅长思考的人寻求帮助。

但是，这种方式不应该只是改变了依赖的对象。

佩特拉说，佛拉基亚的皇帝非常善于利用人。但是，善于利用人的皇帝肯定会考虑如何使用他们。

而那些在高位的人，必须能够成为众人的中心人物。

这就是爱蜜莉娅所选择的目标。

「非常感谢您的鼓舞和支持，爱蜜莉娅。」

背后传来的声音赞美着直视着埃布尔的爱蜜莉娅，她没有转过身去，因为这个声音是她熟悉的，这个人是可靠的奥托。

虽然他此时是在单独行动，而不是在市政厅，但他一定是来找埃布尔的。

「奥托也是来找埃布尔的吗？你一个人没事吧？」

「谢谢你的关心，但是不用担心……」

「是吗？可是奥托，你之前也曾经在大家的视线中被抓走过啊……」

「什么也没发生！……不过，我也有做好防备。」

爱蜜莉娅对奥托的担心让他说话声音更高了，然后加菲尔出现在他的身后。他似乎帮助奥托查找了一些信息。

爱蜜莉娅松了一口气，同时感激地说：

「谢谢你，加菲尔。」

「哈，我不需要谢。比起这个，爱蜜莉、佩特拉小姐你们两个都说了些什么？」

「嗯……很多事情，不过……多亏了佩特拉小姐，我没有被说服。你们有没有被说服呢？」

「嗯，我们没事。非常可靠，让人惊讶呢。」

「呵呵，是吧？」

在奥托和加菲尔面前，爱蜜莉娅和佩特拉击掌庆祝。像加菲尔一样，爱蜜莉娅也为了保护佩特拉而努力。

看着两个人的样子，奥托不禁低下了眼角，但是当他面对「埃布尔先生」时，他又振奋了精神，与对方一起制定战略。

「能猜想出爱蜜莉娅和她们谈了些什么吧。雷姆被九神将带走，前往帝都鲁普加纳，这样一来，我们已经成为帝国的敌对立场了，是这样的话吧？」

「哇！太厉害了！奥托，你偷偷听到了吗？」

「偷听吗？」

「爱蜜莉娅倒无所谓，但就连佩特拉小姐也……我可没有偷听！」

「这样啊。那你还是很厉害嘛。」

「……你真的没有偷听吗，加菲尔先生？」

「爱蜜莉娅倒无所谓，但佩特拉小姐……」

感到惊叹的爱蜜莉娅，和向加菲尔先生确认事实的佩特拉。虽然反应截然不同，但成功猜出他们之前的谈话的奥托苦笑了一下。

接着，奥托望向埃布尔的眼神也变得严肃了起来。

「刚刚听了外面进来的人说，反叛的声音不断在各地响起。人们也在陆续汇聚到这里来了。」

「这在事情的发展和击退『飞龙将』之后就可以想到了。尤尔娜・米希格蕾率领魔都居民并将此地作为根据地，也是必然的结果。」

「是啊，毕竟是令人惊叹的手笔呢。」

埃布尔淡淡地回答着，表示赞同奥托的话。

对于爱蜜莉娅而言，这些内容并不新鲜。在这几天里，聚集在古拉尔的人数很多，爱蜜莉娅也协助修缮着破损的住居。

话虽如此，爱蜜莉娅不擅长力气活儿，所以经常被法兰黛莉卡和佩特拉提醒不要勉强，只是以使用魔法来修补柱子等辅助工作的方式参与。

总之，这些内容对于爱蜜莉娅来说并不是新鲜事。但她感到不安的原因，大概是奥托的态度。

在这个世界中，奥托一如既往地安静、聪明、可靠。但是，爱蜜莉娅感觉到他柔软的侧脸中隐藏着一股无法掩盖的怒火。

「布局到了什么地步呢？」

这句话推进了爱蜜莉娅怀疑的思想，让她更加确信自己的猜测。

埃布尔沉默了。然而，爱蜜莉娅感受到他的目光没有动摇，眼神也没有改变。她认为他没有在思考答案，而是在刻意拖延时间。

「你到底想怎么样？」

奥托用另一种方式问了同一个问题。

爱蜜莉娅并不觉得自己被拖延时间所困扰。她不太喜欢谈论胜负，但在这个场合，胜负已经是注定的了。

奥托的态度，是要问问他失败的原因。然而，他的失败也是爱蜜莉娅等人的问题，他们面临着相同的问题。

「你们的存在，是我们回到这个城市才知道的。如果要考虑到这一点，那就需要预测甚至预言星星的位置。」

「那么，你认为这只是偶然吗？」

「我不打算把它归结为偶然。毕竟，运气只是决定因素之一。」

埃布尔摇了摇头，断然否定了奥托的话。看到奥托的脸色变得严肃，爱蜜莉娅不禁拉了拉他的袖子。这样，她就结束了让他一个人承受失败感的局面。

「奥托，这是怎么回事？」

「……关于夏美小姐现在的处境已经有报告了。虽然觉得这是非常荒唐的事情」

「和米迪娅姆小姐一样，她也变小了，对吧？」

听了奥托的话，佩特拉看了一眼身旁的米迪娅姆。米迪娅姆曾经讲过，在魔都遭到敌人的攻击后，从成年人变成了小孩子。不仅仅是她，昴也处于同样的状况。

为了保护处于不安全状态的昴，

「埃布尔散布了一个谣言，说拥有黑发黑眼的男孩是皇帝的孩子。这样，即使被那些对昴有恶意的人发现，他们也不会杀他......」

「为了确定这件事的真伪，不会拿人命的。当时我也觉得这是一个妙计。不管怎样，只要能降低夏美小姐的危险程度就好了。但是......」

「看来，情况不是只是简单地安全不安全了」

奥托说完，嘴角勾起一抹苦涩的笑容。
