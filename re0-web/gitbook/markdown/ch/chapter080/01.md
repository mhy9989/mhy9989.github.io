# 『胜利条件变更』

------

――佛拉基亚帝国的首都，鲁普加纳成为战场，帝国军与反叛军的攻防战在各地散播着混乱之种，进入了最后阶段。

在位于首都最深处的水晶宮的王座前，佛拉基亚皇帝文森特倒在了光之凶弹之下。在被命令守护首都城墙的『九神将』放弃职责、被认为下落不明的帝国主力部队突然加入战场，以及从西方来的神秘团队在战线上打开巨大空隙的事实，无一不是混乱之种的体现。

然而，在这场战斗中，最为显著地发芽的混乱之种所在――，

「――跟从我吧！现在的局势已经不再是正规军与反叛军的武力冲突！第三方势力的介入改变了胜利条件！」

一个男人站在成为山一般的废墟上，用临时舞台大声喊着。

锐利的黑眸中蕴藏着炽热的理智，干涸的血附在他的黑发上，在战场上回响着轰鸣声，他发出清晰可闻的声音。

在暴力主宰的战场上――不，暴力主宰的佛拉基亚帝国中，这个男人弱小的身姿究竟吸引了什么？

即使想嘲笑他，却没有人嘲笑这个男人。只是将注意力集中，倾听着他的话语。

流淌在他们体内的帝国人的血液，使他们无法置若罔闻这个男人自己定义的存在，在破烂台座上发出的声音。

――从站在废墟上，高声自称皇帝的文森特・佛拉基亚站在那里。

「――――」

众人近距离聆听着文森特高声发出的宣言，那里站着一群显赫人物。

其中包括『九神将』的成员哥兹・拉尔冯和奥尔巴特・邓克尔肯，帝国宰相贝尔斯提兹・冯达尔福，以及作为特殊职位被允许进出皇宫的『星诵者』乌比尔克。

在刚刚过去的攻防战中，考虑到彼此的阵营立场，如此毫无防备地面对面地靠近，实际上是非常危险的。

然而――，

「莫古洛・哈加涅已经控制住局势，但在空中对抗的却是本应死去的巴罗伊・特梅格里夫。不知何种方法，他和自己的爱龙一起复活了。」

「关于这个问题，我有些事情要向陛下报告。――在谒见室里，我遇到了拉米娅・戈德温阁下。她似乎还是以前的样子，本人亲自出现。」

「陛下！这个报告应该是由我来提交的！陛下！不仅是拉米娅阁下，还有在『选帝之仪』中死去的其他皇族也出现了！」

对于文森特的话，贝尔斯提兹和哥兹做出了回应。

尽管哥兹在下落不明的情况下，依然保持着对文森特的忠诚，但贝尔斯提兹的态度转变还是令人不悦。

尽管如此，贝尔斯提兹本来就关心佛拉基亚帝国的繁荣，所以在关乎帝国存亡的情况下，这位老人的态度是自然的。

如果将这些矛盾视为琐事，文森特需要关注的焦点就会变得明确。

「乌比尔克，你事先知道『大灾』的发生。那么，从巴罗伊・特梅格里夫开始，拉米娅和其他皇族的复活，是否就是『大灾』？」

「关于这一点，可以说是『是』，也可以说是『不是』，陛下。我曾听闻文森特・佛拉基亚之死是『大灾』的导火索，为了防止帝国的灭亡，我策划了各种阴谋……」

「所以你对细节一无所知？」

对于这个问题，乌比尔克露出一副轻浮的表情点头说：「是的」。

在某种程度上，这是一个预料之中的答案，但对于文森特来说，这是一个收获。从救助跌落的文森特这一事实来看，乌比尔克和他的追随者『星诵者』与『大灾』是处于对立的立场。

「既然如此，那么现在你的存在并非我所考虑的因素。只要确保自己不会死去，你就好好应对吧。」

「陛下！请允许我来应对这个局面，您务必将自己的安全放在第一位！我哥兹・拉尔冯愿意鞠躬尽瘁，死而后已，来弥补我在您身边缺席的耻辱——」

「——准。」

「——呃」

文森特瞪了一眼充满热血的哥兹，用一句话压制住了他的气焰。

尽管哥兹对皇帝忠诚至上，而且渴望在关键时刻迎头赶上，但事实上，他在从奇夏手中救出文森特之后，已经表现得足够出色。

如果没有哥兹的奋战，文森特很可能会被奇夏俘虏，帝都甚至可能根本无法达到现在的局面。——是的，局面已经形成了。

「————」

文森特将手托住下巴，脑海中勾画出帝都的地图，构思现状。

流失的鲜血和消耗的体力影响着他的思考力，但幸运的是，被奇夏击中锁骨的疼痛让他保持清醒。精神力也是充沛的。

尽管不认为这种疼痛也是奇夏的预谋，但——

「那个，抱歉在大家纷纷扰扰地讨论的时候打扰一下，我有一个吸引全世界目光的任务在身，所以现在可以离开吗？」

「什么！塞西尔斯一将！？你在说什么！？」

「嗯，关于那个一将的事情，偶尔听过几次，但因为兴趣不高，所以完全没留在脑子里。我觉得你可能认错人了吧？如果真的是那样，那位被拿来和我相提并论的一将先生可真是可怜啊！」

在这种情况下，不仅仅是文森特的脑海里快速地运转着思绪。

仍然在数十米范围内——对于莫古洛这样的巨大生物来说，只是一步之遥，与『云龙』梅佐蕾娅和巴罗伊的组合之间的战斗依然在继续。

远远地看到被破坏的蓄水池的水流出，整个帝都被波涛席卷也只是时间问题。此外，闹腾着变小的塞西尔斯也在一旁。

即使哥兹、贝尔斯提兹和奥尔巴特等人都在等待文森特的决定，但这个幼小的『青色雷光』却不受他们的想法影响。

塞西尔斯的外表和内心都显得幼稚，对于文森特的宣言也没有表现出全心全意地信任，

「刚才那段演讲确实很出色！我相信您肯定是佛拉基亚皇帝，但很遗憾，我不会因为某人是皇帝就对他们俯首称臣。如果您想改变我的道路，那就请为我准备一条花路吧！」

「——就像那个菜月・昴所做的一样，是吗？」

「——？很遗憾，我不知道您在说哪位。虽然我觉得他的名字和我认识的某人很像，但我好像和那位美丽的女士讨论过这个话题。」

塞西尔斯歪着头，短短的腿儿踢打着地面回答道。

尽管不知道他关心的那个「美女」是谁，但在幼小的塞西尔斯加入帝都攻防战的时候，文森特就已经确信了。

从西边突袭战场的意外团队，领导他们的正是菜月・昴，而且很可能是他带来了塞西尔斯。

于是——，

「不仅仅是现在，你一直都没有听从我的指示。随你怎么胡闹吧」

「哦，你倒是挺通情达理的。不过，无论你说什么，我本来也没打算乖乖听话，所以请多多包涵。」

「塞西尔斯」

文森特决定放任塞西尔斯，塞西尔斯挥了挥手，矮小的身躯准备跳出去。他的名字被叫到背后，年幼的『青色雷光』回头问：「是？」。

看着那双自从相识以来就一直难以捉摸的眼睛，

「按照当初的约定，我会为你准备一个大展身手的舞台」

「——啊哈」

听到文森特简短的宣言后，塞西尔斯留下了一个小小的笑声，然后消失了。

那风驰电掣般的速度连残影都无法捉摸，如果说这还是未完成的状态，那么佛拉基亚最强的称号就不会被他动摇。不管怎么说——，

「将军，这样做真的没问题吗？塞西尔斯一将……」

「和你们或者乌比尔克的待遇一样，我们现在没有时间顾及这些琐事。比起这个——」

比起处理消失了的塞西尔斯，眼前的问题更加需要应对。

文森特与贝尔斯提兹对视，他用一个足以让哥兹和奥尔巴特等人都能听到的声音说道。

那就是——，

「——放弃帝都。在尽可能保持战斗力的情况下，撤退吧」

在混乱与迷茫的帝都中，一行人骑着红色的疾风马疾驰而过。

队头是众所周知的菜月・昴，左右护卫则是随从骑兵伊德拉・米桑加和唯一的女性，实力强大的坦萨。昴的『普雷阿迪斯战团』的突击力依然强大。

「而且，现在还有碧翠丝和路伊这两大看板，我们正如火如荼！」

正面，握着缰绳的伊德拉的背后，昴短小的手臂抱着碧翠丝，而昴的背后则紧贴着路伊。

不经意间，年幼的四人被成年的伊德拉带领着——，

「――魔枪！」

「呜哇！」

「请恕我无礼」

碧翠丝举手发出的紫黑色箭矢，路伊自由穿梭于空间的攻击，还有坦萨凭借力量将阻碍者击退的强拳，为他们开辟道路。

跃过被摧毁的城墙，昴他们跳进了帝都市区，目标是远处的水晶宫——那座假皇帝驻扎的闪闪发光的城堡。

那个城堡里的假皇帝，欺骗了文森特・佛拉基亚的存在。

只要打败他，就能为昴不得已卷入的佛拉基亚帝国的大内乱画上一个句号。

相信着这一点，昴准备胡乱闹腾地冲了过去。

「施瓦茨！无论到哪里都能钻出来的家伙！到底是什么东西！？」

虽然不能说是惨叫，但伊德拉的声音因根源性的恐惧而发颤。

不知是因为他握缰绳的手传递了动摇，疾风马的步伐也开始产生迷茫。但是，要他们忍受和忘记这种恐惧也太残酷了。

毕竟——，

「到底是什么啊，这些脸色糟糕的家伙们！」

一旦闯进帝都，他们知道帝国士兵的果敢阻挠是理所当然的。

然而，真正在帝都迎击昴他们的帝国士兵仅仅是最初的时候。此后，挡在昴他们面前的都是一群脸色苍白、造型奇怪的存在。

那些脸色青白、眼睛金色的家伙们，全身布满像是玻璃破裂般的裂纹，以一种毫无热情、冷漠的态度阻挠昴他们。

——不仅如此，他们并不只是针对昴他们。

「昴大人，那些家伙，他们也在袭击帝国士兵。」

踢飞青白脸的群体，将他们砸向墙壁的坦萨这样汇报着。

正如她所说，在视线的边缘发生的是，那些『敌人』攻击并非针对昴他们，而是向城市内的守备帝国士兵发起的袭击。

当然，这些『敌人』并不会像『普雷阿迪斯战团』那样，顾及昴的感受。他们挥舞着武器，毫不留情地夺走帝国士兵的性命。

因此——，

「――坦萨！拜托了！」

「我知道，了解。」

听到昴无理的请求，坦萨踢地加速。

她以类似球跳跃的速度冲向敌阵——袭击帝国士兵，追上那些欲夺走生命的一群，用她厚底的鞋扫荡了那些家伙的背影。

「――呃！」

短暂的惨叫响起，被踢飞的『敌人』猛烈地撞向左右的建筑。

被摧毁的建筑，浓烟弥漫，幸存的帝国士兵在那里。坦萨瞥了他们一眼，说了声「快走吧」，让他们离开现场。

幸运的是，被救下的士兵并没有将矛头对准坦萨，而是选择撤离现场。

「这样总算是救了他们……但是，这些家伙」

虽然坦萨的活跃避免了不必要的死亡，但昴却为被她的一击击中、撞向建筑而破碎的『敌人』的惨状而屏息。

『敌人』没有站起来，一部分原因是坦萨的踢击实在太过强力，但更大的原因是他们已无法站起来了。

『敌人』的身体破碎成碎片，变成了四分五裂的状态。然而，这并不意味着他们变成了恶心的尸体，而更像是瓷器杯子掉在地上的破碎方式。

那里没有流淌的血液。就好像他们的体内根本没有血液一样。

「这是什么原理？像是人偶或者机器人一样的……」

「――难道，是『不死王的秘术』吗？」

「――！你有什么线索吗，碧翠丝！」

昴的胸膛托起碧翠丝的身体，她全心全意地信任着他。昴没有错过那双如樱桃般可爱的嘴唇所吐露的话语。

看到昴的反应，碧翠丝微微抬起下巴，

「曾经有一个让死去的灵魂从新的肉体中复活的魔法……也许是禁忌之术吧。这最初是母亲研究的东西，但应该已经因为不完善而被中止了。」

「死者复活的禁术，是不是指死灵法师系的魔法？我脑袋里有些飘飘然，但你母亲可真不是个好东西啊！」

「这不是第一次你说我母亲的坏话了。不过，如果你再这么说，贝蒂也会生气的，你可得小心。」

听到气愤的碧翠丝这样说，昴伸出舌头，闭上一只眼睛表示反省。

然而，可爱的碧翠丝的意见绝不能置若罔闻。如果这种名为『不死王的秘术』的死灵术真的是制造出那个『敌人』的原因的话。

「所以，帝国的『九神将』中有死灵术士之类的家伙吗？」

「……如果是那样的话，埃布尔没事先说这些事情可真有问题。而且，还有区分敌我方面的问题。」

「这在道德观出问题的帝国倒是有可能发生的事……等等，你知道埃布尔的事吗？」

「……我知道。」

碧翠丝的口中蹦出了出乎意料的名字，让昴目瞪口呆。碧翠丝皱着眉头，对昴的惊讶表示肯定。

碧翠丝和埃布尔之间的联系，这不是单纯地名字相同的两个不同人，从她皱眉的表情中可以清楚地看出。

基本上，埃布尔和任何人都不合得来，所以当然和碧翠丝也不合。到底，埃布尔以怎样无礼的态度让碧翠丝痛苦，光是想想，昴的胃就翻江倒海了。

「──这是当然的，那家伙也在这里吗？」

在这场重要的攻防战中，虽然难以想象埃布尔会缺席，但在具体的反叛军编制尚不清楚的情况下，当听到这个肯定时，涌现出了另一种感慨。

如果埃布尔在场，那昴的行动方式也需要一些策略。

「那么，对付僵尸的任务是不是该交给我比较好？还是说，让了解僵尸的可爱碧翠丝和我一起去应对……疼疼疼！什么！？」

「哼，也许吧。只是看到昴的脸有点惹人讨厌。」

「为什么！？这憨态可掬和狂妄自大之间的眼神可是天生的！？」

「听到埃布尔的名字，你一副松了口气的样子让我生气。」

对于碧翠丝不合理的指责，昴被捏了大腿，嘴角勾成了个倒「へ」。

虽然无法确切地说，但在稍稍分开的这段时间里，碧翠丝看待昴的眼神似乎变得严厉了。

诚然，他承认埃布尔在能力上优秀，但在这场大规模战乱的情况下，听到埃布尔的名字并不会感到安心。

无论如何，与碧翠丝重逢比埃布尔要让人愉快得多。

「对我来说，你才是最重要的，碧翠丝，别误会。」

「那么，你要在今后证明这一点。」

「嗯，当然了！」

为了回应碧翠丝的感情判断，昴拍了拍伊德拉的肩膀，让疾风马在原地停下。

周围既没有敌人的影子，也没有平民，但他想在继续前行之前确定一下接下来的方针。

「施瓦茨大人，如果您的玩闹已经结束，那接下来打算怎么做呢？」

「真是刻薄！我并不是在玩闹……不过，碧翠丝！那个僵尸魔法，如果击败了施法者，它会停止吗？」

在被坦萨那毫无表情的侧脸刺激的同时，昴向碧翠丝寻求详细的意见。碧翠丝被问及此事，一边用手指扯着自己的卷发，一边回答道：

「干预他人肉体的魔法主要分为阴魔法和阳魔法。无论哪种属性，基本上施法者倒下后，魔法效果会中断，这是一般规律。所以，我想这个也应该是如此。」

「这话里带着犹豫啊。不是很确定吗？」

「贝蒂也是第一次见到『不死王的秘术』的实现！我不能以猜测来断定这件事，那样太像孩子般的不负责任了！」

「冷静冷静！那么，魔法的有效范围和施法地点等方面也无法确定吗？」

「……虽然很遗憾，但这还是取决于使用者。」

碧翠丝似乎担心会再次被坦萨讽刺，带着遗憾地看了她一眼。然而，承认力量不足的碧翠丝并没有被坦萨责备。

正因如此，即使对方不是易相处的人，坦萨也不会贬低她们。

「现在不是争论的时候，这对我们很有帮助。碧翠丝，谢谢。」

「虽然被表扬了，但并不是一个好答案。」

在一边说着感谢的话，一边抚摸着头时，碧翠丝的脸颊鼓了起来。虽然有想用手指戳戳她鼓起的脸颊的冲动，但他感觉这是一个困难的局面。

从情况来看，这次僵尸袭击显然是个异常情况。

从帝国士兵遭到袭击来看，虽然有可能是反叛军中有使用死灵术的人，丧失道德观念和人性的埃布尔发动了这种残忍的战术，但僵尸出现的方式和数量令人在意。

这些僵尸显然不是从城市外部涌入市区的，而是在城市内部产生并四处肆虐。

如果这是埃布尔的策略，从一开始就发动消耗战会更有利。或许，昴不知道的战术正在被使用。

「果真，如果用到僵尸的话，我得重新考虑我们的关系。」

而且，虽然只是一种直觉，但昴觉得埃布尔或许会在政治上利用『死亡』，但不会把『死者』当作战力来使用。

无论如何都无法避免佛拉基亚式观念的埃布尔，也许是不允许这样的行为。

因此——

「这个僵尸大师的存在应该是个意外。如果现在就冲进城堡打倒假皇帝，战争还没有结束的话……」

「那么，施瓦茨，我们该怎么办？我们会听从您的指示。」

「伊德拉……」

昴把手搭在下巴上，伊德拉扭过头来，用言语表达了信任。

这并不是他把责任推给昴，而是在可能会面临生死关头的情况下，表示愿意和昴一起拼命。

昴自愿地站到了这个位置。

他必须承担这个责任。而且，多亏了伊德拉和普雷阿迪斯战团的众人，昴现在能发挥的作用比以前更大。

「虽然缺乏决定性的证据，但这个地方还是……啊啊啊啊！疼疼！」

「昴！？」 「施瓦茨大人！？」

昴对责任的自觉表示赞同，但是他因为别的痛苦尖叫起来，这让碧翠丝和坦萨都吃惊不已。

这一次，昴的痛苦是因为头发被扯了。而扯他头发的人，既不是碧翠丝，也不是坦萨，甚至不是伊德拉，而是——

「什么，路伊！别扯我的头发！从小就不忽视护发，这是为了预防变老后脱发的防御策……」

「哇哦！哦！啊哦！啊——哦！」

「怎么了？」

在昴用手护着被扯疼的头皮时，路伊拼命地向他诉说着什么。

她拍打着昴的背，然后指向帝都的北方——水晶宫所在的方向，但又稍稍偏离，不断地提醒着他。

她的意思是让他朝那个方向走，这一点他明白。

「那边有什么？」

「啊哦！」

对昴的问题，路伊高声回答。

虽然这么说可能有些怪，但与昴汇合后的路伊在精神上显得相当稳定。昴虽然与她有复杂的关系，但对于她的平安还是由衷地感到欣慰。

既然如此，如果路伊如此迫切地呼唤着某件事情的话——

「——碧翠丝，告诉我一件事。大家都在一起吗？」

「……并不是所有人都在一起。」

在昴安静地提问时，碧翠丝的回答刻意回避了明确的表述。

但是，她这样做并非是在故弄玄虚，而是为了避免让昴受到打击。昴明白了这一点，也明白了她为什么要含糊其辞。

所以——

「坦萨、伊德拉！还有碧翠丝和路伊，我们要改变策略！」

将要做的事情整理好后，昴向以疾风马为中心的团队成员喊话。

很可能，以帝都为中心的战斗不会平息，而是朝着更加激烈的方向发展。鉴于僵尸及其 Master 这种难以预料的存在，他们必须有所准备。

当然，可以列出这样的看似合理的理由，但——

「首先，我们要去路伊指向的那个方向！让我们在那里捡到重要的东西！」

「路伊的……那么，她在那边吗？」

从昴的决定中，碧翠丝也了解了路伊所指示的方向所存在的东西。

对碧翠丝的话表示同意后，昴转向注视着他的坦萨和伊德拉，继续说道。

那就是——

「——我们要找到雷姆，带着尽可能多的人从帝都逃跑！这是一场与时间的赛跑！」

如此决定，与在场的人们意外地指向了相同的方向。
